# Weather Voice Agent

This repository implements a simple voice assistant that tells weather for a requested city.

## Overview

- Voice input/output using the browser (Web Speech API).
- Weather data using a public weather API (e.g., OpenWeatherMap).
- Error handling for:
  - Invalid / unknown city.
  - API errors.
  - Unclear speech or no speech detected.

## Project Status

This repo is being built step-by-step for an internship assignment.
More details will be added as features are implemented.
